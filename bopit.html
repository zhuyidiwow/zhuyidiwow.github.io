Bop It Extreme Home Edition

General Information

Project Duration 1 week
Team Size 5
My Contributions UI Design, Programming, Wood work
Theme A fun-focused virtual world in a week
Genre Competitive local 2-player arcade machine
Platform Makey Makey + Handmade machine

Brief Description

The objective of this assignment is to focus on the fun part of a game and make a highly fun, engaging activity. So we decided to extend the original Bop It game and make it a local 2-player competitive experience with large scale machines.

Based on the idea, we designed and built the brand new pair of machines with parts available from ETC's workshop and purchased from construction waste reuse stores.
